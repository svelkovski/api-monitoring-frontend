# API Monitoring Tool - Frontend

This is the Angular frontend for a developer-focused API monitoring tool.  
It provides dashboards, API detail views, and interaction with the backend REST API.

## Technologies Used
- Angular 18
- Tailwind CSS
- RxJS for reactive data

## Features
- Login / Registration pages
- Dashboard with API status and uptime
- API detail page with response history charts
- Add / edit / delete monitored APIs
